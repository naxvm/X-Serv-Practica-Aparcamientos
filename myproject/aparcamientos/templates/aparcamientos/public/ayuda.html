Práctica de SARO (2017): Aparcamientos

Autor:Ignacio Condés Menchén
Sitio web para aglutinar a partir de datos del XML del ayto. de Madrid (enlace en el pie de página), analizando el mismo y actualizando una base de datos de sus aparcamientos en el servidor.
Este tiene soporte para usuarios, los cuales pueden seleccionar aparcamientos para añadirlos a su página personal, además de personalizar el color de fondo y tamaño de fuente que visualizará en toda la página.
En la página de cada aparcamiento se podrá añadir un comentario (anónimo) al mismo.
Además, se podrá visualizar solamente los aparcamientos cuyas instalaciones sean accesibles, marcando dicha opción al mostrarlos.
